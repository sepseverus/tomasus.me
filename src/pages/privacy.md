## Analytics

This site uses [GoatCounter](https://www.goatcounter.com/) for privacy-friendly analytics. GoatCounter:

- Does not use cookies
- Does not track personal information
- Does not collect IP addresses
- Is GDPR-friendly by design
- Provides only aggregate, anonymous statistics

No tracking scripts from Google, Facebook, or other third parties are used on this site.

## Comments

This site does not currently collect comments. If commenting is added in the future, it will use privacy-respecting solutions like [Giscus](https://giscus.app/) (GitHub Discussions-based), which requires explicit user consent.
